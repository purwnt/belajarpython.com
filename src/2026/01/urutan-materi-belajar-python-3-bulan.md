Belajar pemrograman, khususnya Python, bisa terasa membingungkan jika tidak memiliki peta jalan (roadmap) yang jelas. Banyak pemula terjebak dalam tutorial tanpa arah (*tutorial hell*) dan akhirnya merasa tidak ada kemajuan.

Jika Anda serius ingin menguasai Python dalam waktu 3 bulan, Anda membutuhkan kurikulum yang terstruktur. Berikut adalah urutan materi belajar Python yang dirancang khusus untuk membawa Anda dari nol hingga mahir dalam 90 hari.

## Jadwal Belajar Mingguan

Berikut adalah kurikulum terstruktur yang bisa Anda ikuti selama 12 minggu:

| Minggu | Fokus & Topik | Materi Detail |
| :--- | :--- | :--- |
| **Bulan 1** | **Dasar-Dasar & Logika** | *Fokus: Memahami sintaks dan logika pemrograman* |
| 1 | Persiapan & Sintaks | • Instalasi Python & Code Editor (VS Code)<br>• Program "Hello World"<br>• Variabel & Tipe Data (String, Integer, Boolean)<br>• Operasi Aritmatika & Perbandingan |
| 2 | Kontrol Alur (Flow) | • Percabangan `if-else` untuk pengambilan keputusan<br>• Perulangan `for` dan `while` loop<br>• Mengontrol loop dengan `break` & `continue` |
| 3 | Fungsi (Functions) | • Membuat fungsi dengan `def`<br>• Parameter & Argumen<br>• Mengembalikan nilai (`return`)<br>• Scope Variabel (Lokal vs Global) |
| 4 | Latihan Algoritma | • Studi Kasus: Membuat Kalkulator Sederhana<br>• Studi Kasus: Game Tebak Angka<br>• Studi Kasus: Konversi Suhu |
| **Bulan 2** | **Struktur Data & OOP** | *Fokus: Mengelola data kompleks & menulis kode rapi* |
| 5 | Struktur Data | • **List & Tuple**: Data berurut<br>• **Dictionary**: Data key-value<br>• **Set**: Data unik<br>• List Comprehension |
| 6 | File & Error Handling | • Membaca file (`.txt`, `.csv`)<br>• Menulis ke file<br>• Menangani error dengan `try-except` |
| 7 | Dasar OOP | • Konsep Class & Object<br>• Attributes & Methods<br>• Inheritance (Pewarisan sifat class) |
| 8 | Module & Library | • Import Module bawaan (`math`, `random`)<br>• Menggunakan PIP untuk install library eksternal<br>• Membuat Virtual Environment |
| **Bulan 3** | **Spesialisasi & Proyek** | *Fokus: Penerapan ilmu ke dunia nyata* |
| 9 | Library Populer (Pilih Fokus) | • **Web Dev**: Dasar Flask / Django<br>• **Data Science**: Dasar Pandas / Matplotlib<br>• **Automasi**: Selenium / PDF Processing |
| 10 | Git & GitHub | • Menyiapkan Repository<br>• Konsep Commit, Push, dan Pull<br>• Membangun portofolio kode di GitHub |
| 11 | Proyek Capstone | Kerjakan satu proyek besar:<br>• Web: Aplikasi To-Do List dengan Database<br>• Data: Dashboard Analisis Penjualan<br>• Auto: Bot Pengirim Email Otomatis |
| 12 | Finalisasi | • Code Refactoring (Merapikan kode)<br>• Deployment (Upload aplikasi ke internet)<br>• Review ulang materi yang sulit |

## Tips Agar Konsisten Selama 3 Bulan

1.  **Koding Setiap Hari**: Luangkan minimal 30-60 menit setiap hari. Konsistensi mengalahkan intensitas.
2.  **Jangan Hanya Menonton**: Tutorial video bersifat pasif. Anda harus mengetik ulang dan memodifikasi kode.
3.  **Gabung Komunitas**: Bertanya di forum seperti StackOverflow atau grup Telegram Python Indonesia jika macet.

Dengan mengikuti urutan materi ini, dalam 3 bulan Anda bukan lagi seorang pemula yang bingung, melainkan seorang programmer Python yang siap mengerjakan proyek nyata. Selamat belajar!
